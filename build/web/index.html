Servlet Interface

This is implemented by using Servlet Interface.We need to @Override these following methods:

  • init(ServletConfig)
  • service(ServletRequest, ServletResponse)
  • destroy()
  • getServletConfig()
  • getServletInfo()
Go and Check

Generic Servlet Class

This is implemented by using GenericServlet class.We only need to @Override these following method:

  • service()

Other all methods body is already defined in this Generic
class becouse it can implementes Servlet Interface internally.

Go and Check

HttpServlet

This is implemented by using HttpServlet.This can provide methods for particular protocol.Like:

  • doGet(HttpServletRequest req, HttpServletResponse res)
  • doPost(HttpServletRequest req, HttpServletResponse res)
  • doPut(HttpServletRequest req, HttpServletResponse res)
  • doTrace(HttpServletRequest req, HttpServletResponse res)
  • doHead(HttpServletRequest req, HttpServletResponse res)
  • doDelete(HttpServletRequest req, HttpServletResponse res)
  • doOptions(HttpServletRequest req, HttpServletResponse res)
